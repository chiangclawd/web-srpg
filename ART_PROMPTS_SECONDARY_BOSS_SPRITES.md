如果是新對話，先重貼 `ART_PROMPTS_BATTLE_SPRITES.md` 的「🎬 第一步：風格錨定」段落，再貼下面這段：

## 🔄 不滿意時的修圖指令

### 氣場太強（搶主要 BOSS 風頭）
```
保持構圖跟畫風，但把腳下/馬蹄下的氣場霧氣再淡化 50%，要變成「幾乎看不到、只有邊緣有色彩」的程度。
```

### 氣場太弱（跟雜兵差不多）
```
保持構圖跟畫風，但把氣場霧氣再加強一點點，要在戰場上一眼能看出「這是 BOSS 不是雜兵」。
```

### 卡爾將軍 / 莫頓的馬太大或太小
```
保持構圖跟風格，但讓馬的比例調整成佔畫面高度的 60%（騎士佔 75%），讓兩者比例平衡。
```

### 賽琳 / 凜法師感不足
```
保持構圖，但讓書本上方的法術霧氣再明顯一點，腳下也要有對應顏色的微弱魔法陣（極小、模糊）。
```

### 副教主・凜畫得太兇
```
保持構圖跟服裝，但讓表情更柔和、嘴角放鬆一點，眼神要「悲傷而非仇恨」。她不是典型反派。
```

### 弗拉德笑得太誇張
```
保持構圖，但收一點嘴角的笑容、犬齒不要露太多，要「冷笑」而不是「狂笑」。
```
