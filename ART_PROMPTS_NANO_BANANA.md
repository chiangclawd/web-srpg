## 📌 第一步：風格錨定（最重要！只貼一次）

**新開一個對話，第一句先貼這段**：

```
我接下來要請你連續產出一系列日式 JRPG 風格的角色立繪，總共 10 個角色。

風格基準（每張都要符合）：
- 動漫 JRPG 立繪風格，類似於《天使帝國 II》、《Tales of 系列》、《聖火降魔錄 火紋》GBA 時代的角色設計
- 柔和賽璐珞著色（cel shading）+ 軟陰影
- 線稿乾淨清晰、平塗主色 + 簡單漸變
- 飽和度中等（不過度鮮豔，不死灰）
- 統一光源：左前方上方 45 度
- 構圖：胸部以上的半身像、微微側面
- 背景:純白色或非常淡的灰白漸層,角色清楚分離以方便去背
- 解析度:1024 × 1024 正方形
- 不要文字、簽名、浮水印、多人

請確認你理解這個畫風指引,接下來我會給你 10 個角色的詳細描述,每個都要保持完全一致的線條粗細、上色技法、光影風格、頭身比例。

確認後請回我「了解,開始」,然後等我給第一個角色。
```

等 Nano Banana 回「了解,開始」後,**才**開始貼下面的角色 prompt。

## 🛠️ 修圖指令範本（產出後不滿意時用）

Nano Banana **最強的優勢** = 可以直接編輯。常用句型：

### 改外觀
```
保持整體構圖跟畫風不變,但把亞瑟的頭髮改短一點。
```
```
保持這張臉跟身體不變,把斗篷顏色從白色改成淺金色。
```

### 改表情
```
其他都不變,把表情改成微笑,眼神更溫柔。
```
```
其他都不變,把蘿莎的表情改成大笑,看起來更活潑。
```

### 改背景
```
把背景改成完全純白色,角色周圍邊緣清楚分離。
```
```
把背景換成柔和的天藍色漸層。
```

### 去背
```
把背景完全去除變成透明 PNG,只保留角色本身。
```
> ⚠️ Nano Banana 不能直接出透明 PNG,但可以「純白背景」之後再用 [remove.bg](https://remove.bg) 去背。

### 統一風格
```
這張看起來線條比前面幾張粗,請重畫一次,線條粗細跟前面 4 張一致。
```

### 多視角(進階)
```
保持亞瑟這張臉跟服裝不變,生一張他在戰場上揮劍的全身動作姿勢圖。
```

## 🚦 Nano Banana 常見問題

### Q1: 同一個角色我重新生成,風格飄了
**A**: 把上一張**圖丟回去**對它說「保持這個畫風重畫,我要的姿勢是 [...]」
> Nano Banana 看到圖就會跟著一致

### Q2: 連續生 5 張後,第 6 張突然線條變粗
**A**: 它「忘了」你最初設定的風格。對策:
- 把第 1 張(風格基準)再丟一次當參考
- 或開新對話、重貼風格錨定

### Q3: 角色臉一直崩
**A**: 加入「symmetrical face, both eyes clearly visible, anatomically correct」
> 或者切換成英文 prompt 試試,Nano Banana 對英文有時更穩

### Q4: 背景去不乾淨
**A**: 不要在 Nano Banana 強求,用 [remove.bg](https://remove.bg) 一鍵去背最快

### Q5: 解析度不夠 / 細節模糊
**A**: Gemini 2.5 Flash Image 預設輸出 1024x1024 已足夠 HubScene 用(只顯示 88x88)
> 之後若要 Cutscene 大圖,用「上採樣到 2048」可能要外接 [ImgUpscaler](https://imgupscaler.com)

### Q6: Gemini 付費版 vs 免費 quota
**A**: 你已付費,quota 對你不是問題,放心連續產出。每張約 5-10 秒。
