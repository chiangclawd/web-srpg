# 🛡️ 王國泛用兵種戰場 Sprite — Nano Banana Prompt

> 為 5 位玩家陣營雜兵（王國劍兵 / 王國槍兵 / 騎兵斥候 / 王國弓兵 / 見習法師）做專屬戰場 sprite。
>
> **程式已就緒**：assetManifest.ts 已加 `sprite_player_<unitType>` 條目，
> 並把 sprite 解析邏輯改成 player 也會 fallback 到泛用兵種圖（之前只有 enemy 會）。
> 現在所有沒有專屬 sprite 的 player 雜兵都會找 `sprite_player_<unitType>`，
> 找不到才回到原本的色塊；放幾張就生效幾張。

---

## 📐 規格（與其他 sprite 完全一致）

- **尺寸**：256×256 PNG，**透明背景**
- **路徑**：`public/assets/sprites/<檔名>.png`
- **構圖**：全身、面向右（程式自動翻轉）、雙腳踩平地
- **風格**：Q 版（chibi/SD），頭身 1:2 ~ 1:2.5

## 🎬 風格錨定（沿用之前的 sprite 對話）

**最佳作法**：在你之前產敵方泛用 6 兵種 sprite 的同一個對話接續貼。Nano Banana 已建立泛用兵種記憶。

如果是新對話，先重貼 `ART_PROMPTS_BATTLE_SPRITES.md` 的「🎬 第一步：風格錨定」段落，再貼下面這段：

```
接下來 5 個是「王國泛用雜兵」（玩家陣營），跟之前產的「敵方泛用 6 兵種」是同一個層級
（雜兵級、無臉、可重複使用），只是改成王國軍配色：

- 配色主軸：王國藍（深寶石藍 / 銀灰）+ 金色裝飾線（皇室紋章配色），跟敵方的暗紅 + 深灰相對
- 統一胸前/盾面/披風 配王國的金獅紋章（小巧、不要喧賓奪主）
- 服裝整潔（王國正規軍是訓練有素的部隊，比草寇式敵兵更整齊）
- 頭盔/兜帽遮住臉部（無臉雜兵原則，但比敵方士兵的全罩頭盔多一點點面部留白，象徵王國軍紀律但人性化）
- 沒有個人氣場霧氣（這是雜兵不是 BOSS）
- 表情中性（雜兵不要有強烈個性）

請確認你理解，接下來我會給你 5 位的詳細描述。
```

## 1️⃣ 王國劍兵（player_sword）

```
單位：王國劍兵泛用。

維持 Q 版 JRPG 戰場 sprite 風格。這是王國正規軍劍兵雜兵，不要強調個人特徵。

設定：
- 性別：男性無臉雜兵
- 服裝：銀灰色板甲胸甲（胸前有金色小獅紋徽章），深寶石藍色的鎖子甲下擺與護腿，深褐色皮帶
- 頭盔：銀色開放式半盔（露出下半張臉但是中性的、無個人辨識特徵），盔頂有藍色短羽飾
- 配件：右手握直劍（劍身銀灰、劍格金色），左手握圓盾（盾面深藍配金色獅徽）
- 姿勢：正面戰鬥準備姿勢，雙腳分開站穩
- 體格：標準成人男性，看起來訓練有素

256x256，透明背景，面向右。要能在 64x64 縮圖也能辨識「王國劍兵」職業（藍金配色）。
```

## 📋 步驟（每張獨立做）

```
1. 開 aistudio.google.com，選 Gemini 2.5 Flash Image
2. 接續敵方泛用 sprite 的對話（最佳）；或新對話先重貼風格錨定 + 上方「王國泛用兵種額外指引」
3. 依序貼 1️⃣ ~ 5️⃣ prompt（建議劍 → 槍 → 弓 → 法 → 騎，最後做騎兵因為馬最複雜）
4. 不滿意 → 對話 Nano Banana 微調（見下方）
5. 滿意 → 右鍵存成對應檔名：
   player_sword.png, player_lance.png, player_cavalry.png, player_archer.png, player_mage.png
6. 全部放到：public/assets/sprites/
7. push 即生效（manifest 已預先 wire 好，不需要再改程式）
```

## 🔄 不滿意時的修圖指令

### 太像個人英雄（缺乏雜兵感）
```
保持構圖跟畫風，但去掉所有個性化細節（特殊髮型、表情、紋身），讓他更像「一張可以重複貼上的軍隊員工」。
```

### 王國配色不夠統一
```
保持構圖，但把藍色再深一點（深寶石藍而非天空藍），金色徽章再小一點但更精緻（不要太花）。
```

### 騎兵的馬太搶眼
```
保持構圖，但讓馬的顏色再深沉一點（暗棕色），讓視覺重心仍在騎士身上。
```

### 弓兵頭髮露太多
```
保持構圖，但把皮帽戴得更低，讓頭髮只露出極少一點（雜兵不要強調個人特徵）。
```

### 見習法師太老
```
保持構圖跟服裝，但讓臉龐更年輕一點（見習生才剛出魔法學院，不是熟練法師）。
```

### 王國 vs 敵方雜兵看起來太像
```
保持構圖，但加強配色對比：王國劍兵的胸甲銀色再亮一點、藍色再藍一點、金色獅紋再清晰一點，
跟敵方的暗紅+深灰要在 64x64 縮圖時也能一眼分辨。
```
