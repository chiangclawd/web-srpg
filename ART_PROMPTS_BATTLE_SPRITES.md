# ⚔️ 戰場單位 Sprite — Nano Banana Prompt

> 為戰場上的單位（取代色塊形狀）生 Q 版／全身像 sprite。
> 程式已就緒：放到 `public/assets/sprites/<檔名>.png` 並取消 manifest 註解即生效。

---

## 📐 規格

- **尺寸**：256×256 PNG（正方形，全身佔畫面 80% 高度）
- **路徑**：`public/assets/sprites/<檔名>.png`
- **去背**：必須透明背景（不要白底！）
- **構圖**：**全身正面或微微側身**，腳踩平地（畫面下緣）
- **方向**：**面向右**（程式會自動水平翻轉敵方面向左）

---

## 🎨 風格定位

跟立繪不同！立繪是「臉部特寫的胸像」，sprite 是「戰場小人」。
推薦風格：**Q 版（chibi）/ SD（super deformed）**

```
頭身比例 1:2 ~ 1:2.5（頭比較大，可愛戰場感）
身體簡化但保留標誌性裝備特徵
類似於：天使帝國、聖火降魔錄 GBA、Octopath Traveler 2D 風
```

## 🔄 不滿意時的修圖指令

### 角色辨識度不夠
```
保持構圖跟畫風不變，但讓亞瑟的金色頭髮更亮、銀色胸甲反光更明顯，這樣縮小到 64x64 也能一眼認出是他。
```

### 整體太繁複
```
保持風格，但簡化細節：減少裝甲上的紋路、減少斗篷的皺褶。讓畫面在縮小後仍然清楚。
```

### 角色不夠 Q 版
```
保持構圖跟畫風，但把頭再放大一點（頭身比例改 1:2），眼睛也再大一點。要更明顯的 chibi 風格。
```

### 背景沒去乾淨
```
保持角色不變，把背景完全替換成透明（PNG alpha 透明）。確認沒有白色或灰色背景殘留。
```

### 敵方雜兵太個性化
```
保持構圖，但讓臉部完全被頭盔/兜帽遮住，不要露出太多五官。雜兵不需要有個性。
```

## ⚠️ 重要技術提示

### 為什麼 sprite 要面向右側？
程式自動處理：
- 玩家陣營（faction='player'）→ 不翻轉 → 面向右（衝向敵方）
- 敵方陣營（faction='enemy'）→ 自動水平翻轉 → 面向左（衝向玩家）

所以你只需要畫**一個方向**（面向右），程式幫你處理另一個方向。

### 為什麼建議 256×256？
- 戰場 hex 寬約 80px，sprite 會被縮到約 64×64 顯示
- 256×256 提供足夠細節，縮小後仍清晰
- 比 1024×1024 立繪小很多，載入更快（手機友善）

### 透明背景檢查
存檔後用預覽程式打開，確認背景是「方格圖案」（透明）而不是白色。
若不是透明，用 Finder Quick Actions → 移除背景。
