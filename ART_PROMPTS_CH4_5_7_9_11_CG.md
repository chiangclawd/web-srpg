## 📐 共通規格（每張都要符合）

- **尺寸**：1920×1080 PNG（16:9 橫式）
- **檔名**：`chapterN.png`（N = 章節數）
- **路徑**：`public/assets/cg/chapterN.png`
- **構圖**：右下角留白給對話框（**畫面下方 1/3** 是對話框會壓住的區域，重要元素別放那）
- **使用方式**：cutscene 入場全螢幕背景 + 35% 黑罩 + 立繪坐在對話框上
- **無人物臉部、無文字、無浮水印**（角色用立繪）

## 🔄 不滿意時的修圖指令

### 整體太亮 / 太暗
```
保持構圖跟畫風不變，把整體色調調暗 15%，但保持光源細節（陽光、月光、火焰）的對比度。
```

### 視覺重心不對
```
保持風格，但讓視覺重心往左偏移，右下角留更多空白（給對話框用）。
```

### Ch4 霧不夠
```
保持構圖，但讓晨霧再厚一點、河面倒影再朦朧一點，整體再帶點冷藍調。
```

### Ch5 城牆太完整 / 太破
```
保持構圖，但讓北門再多一些破壞痕跡（焦黑、撞錘凹陷），但城門還沒被攻破。
```

### Ch7 虛空裂痕不夠暗示
```
保持構圖，把雲層中央的虛空裂痕區域再扭曲一點、紫紅光暈再強一點，但不要畫出具體形狀。
```

### Ch9 廢村感不足
```
保持構圖，但讓廢村建築再多一些「曾經有人住」的細節（晾衣繩、孩童塗鴉、舊招牌），讓血祭祭壇與廢村的對比更強烈。
```

### Ch11 七光柱過於工整
```
保持構圖，但讓七道光柱高低不一、傾斜方向不同，避免規律對稱，更有「七人各自集結」的感覺。
```
