## 5️⃣ 副教主・凜（Sub Priest Lin）— 黯的悲劇師妹

```
角色 15：副教主・凜。

維持 JRPG 動漫立繪畫風。這是第 9 章（番外）BOSS，黑暗教派副教主，教主・黯的師妹。雖然是反派，但她的悲劇在於——她對師兄黯有未說出口的愛戀，五十年來只為了讓他「能再活一次」而墮入黑暗。氣質要「美麗而悲傷」，不要兇惡。

設定:
- 性別年齡：女性，外表 25 歲（實際 50+，被黑暗教派的長壽法術延緩老化）
- 髮型髮色：淡銀紫色長直髮（直至腰部），髮稍略捲，前額有齊瀏海，左側用一支銀色月牙髮簪固定一束髮
- 眼睛：清澈的淺紫色，眼神帶著揮之不去的憂愁，眼睫毛特別長
- 臉型：精緻的瓜子臉，皮膚白皙近乎透明（長期不見陽光），右眼角下方有一顆淚痣
- 服裝：深紫黑色法袍（不像主教那樣暗紅），袍領和袖口繡銀色月光紋路，胸前懸掛一個小銀月吊墜（與教主・黯的虛空徽章成對比，暗示她代表「月光」而非「虛空」）
- 配件：雙手抱著一本古老法術書（書面是冰藍與銀白配色，封面有月亮浮雕），書頁上方飄出淡紫色與銀色的霧氣
- 表情：嘴角微微抿著（不笑也不怒），眼神有距離感，像是在思念遠方的人——而不是仇恨

胸部以上立繪。背景純白偏冷紫色淡漸層（與教主・黯的深黑紫色形成對比）。
1024x1024。維持與夏倫類似的優雅法師氣質，但更陰沉、更悲傷。

關鍵：她的氣質要「美麗而悲哀」，不能畫成典型反派，否則第 9 章的劇情會失味。
```

## 🔄 不滿意時的修圖指令範本

### 凜的氣質太兇 → 改溫柔
```
保持整體構圖、畫風、髮色不變，把凜的眼神改成更溫柔、更悲傷的，嘴角微微下垂一點。讓她看起來像是在思念某個人，而不是憎恨。
```

### 卡爾不夠瘋狂
```
保持整體構圖跟畫風不變，把卡爾的眼神改得更狂熱，瞳孔更小、眼下黑眼圈更深一點。表情從介於微笑和咆哮，調成更明顯的扭曲。
```

### 弗拉德太斯文
```
保持構圖跟畫風，但把弗拉德的氣質調得更野蠻，臉部表情更兇狠，狼皮披肩的狼頭再凸顯一點。
```

### 莫頓太「強」了 → 應該更愚笨
```
保持構圖跟畫風，但讓莫頓看起來更「不聰明」一點：眼神可以亢奮但不要有計謀感，表情更直接、像是只會衝鋒的暴徒。
```

### 賽琳的冷感不夠
```
保持構圖跟畫風，但把賽琳的整體色調再調冷一點，皮膚再蒼白，背景的藍色淡漸層更明顯。讓她看起來像是長期住在雪原。
```
