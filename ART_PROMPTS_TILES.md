# 🎨 戰場地形 Tile — Nano Banana Prompt

> 為戰場 4 種地形產 12 張 painterly anime hex 形圖塊（每種 3 變體散布打散重複感）。
> **程式已就緒**：`assetManifest.ts` / `BootScene.ts` / `BattleScene.drawBoard()` 全 wire。
> Pipeline：把 PNG 丟進 `public/assets/tiles/<key>.png` push 即生效；缺檔仍會回到色塊版。

---

## 📐 共通規格

- **形狀**：**pointy-top hex**（六邊形，尖角朝上下）
- **尺寸**：138 × 160 PNG（2x，遊戲內顯示 69×80）
- **背景**：**完全透明**（hex 邊外的四角必須 100% transparent，否則會疊出方框）
- **無文字、無邊框、無浮水印**（程式會在 hex 邊緣自動畫格線）
- **風格**：painterly anime — 賽璐珞著色 + 戲劇性光影 + 飽和度中等，跟既有立繪 / CG 同 family
- **無人物無單位**（純地形材質）

---

## 🎬 共通風格錨定（新對話只貼一次，12 張共用）

```
我接下來要請你連續產出 12 張 JRPG 戰場 hex 地形圖塊，每張都要符合這個風格：

風格基準：
- 138×160 pointy-top hex 形（尖角朝上下，**hex 邊外的 4 個角必須完全透明**）
- painterly anime 風格 — 賽璐珞著色 + 柔和光影
- 飽和度中等、色塊明確、有手繪溫度，**不要 3D 渲染感**
- 沒有人物、沒有文字、沒有邊框（程式會自動畫格線）
- 「俯視角」45 度視角，能看出地形特徵但不是純頂視

每種地形要產 3 個變體（細節略不同但同色系、同風格），讓地圖中相鄰相同地形不會像複製貼上。

請確認你理解，接下來我會給你 4 種地形 × 3 變體 = 12 張的詳細描述。

確認後請回我「了解，開始」，然後等我給第一種地形。
```

等回「了解，開始」後才貼下方 prompt。

## 1️⃣ 平原（plain）— 3 變體

> 主色調：暖調晨光草地、黃綠 + 米色高光。檔名：`plain_1.png` / `plain_2.png` / `plain_3.png`

```
地形 1：平原 × 3 變體。

主題：晨光下的開闊草地

風格：
- 暖調陽光黃綠色草地（不是陰沉森林綠，是「陽光下的草坪」）
- painterly 筆觸可見、不是平塗

變體 1（plain_1）：純草地
- 一片均勻的草地，少量小白花散布、有極淡的「風吹過」筆觸感
- 主色：黃綠色 + 米白小花高光

變體 2（plain_2）：泥土小徑
- 草地中間斜過一條淺褐色泥土小徑（不是直線，是自然彎曲）
- 主色：黃綠 + 淺褐土徑

變體 3（plain_3）：碎石草地
- 草地中散落幾顆灰色岩石（小、分散、不規則）
- 主色：黃綠 + 灰岩

三張要保持一致的草地色調、光源方向（左上 45 度）、筆觸風格，只是局部細節不同。

每張 138×160 pointy-top hex，hex 邊外完全透明。
```

## 2️⃣ 森林（forest）— 3 變體

> 主色調：藍綠調針葉林、深陰影。檔名：`forest_1.png` / `forest_2.png` / `forest_3.png`

```
地形 2：森林 × 3 變體。

主題：濃密針葉林（俯視角能看到樹冠）

風格：
- 比平原暗 + 比平原藍（藍綠調，跟平原黃綠拉開）
- painterly 樹冠堆疊感

變體 1（forest_1）：均質針葉林
- 整個 hex 是一團濃密的深藍綠樹冠（俯視看不到地面），有明暗筆觸體現「樹群」感
- 主色：深藍綠 + 局部高光

變體 2（forest_2）：林中空地
- 中央一小塊縫隙能看到地面（深棕色腐葉），周圍仍是樹冠包圍
- 主色：深藍綠 + 中央深棕

變體 3（forest_3）：邊緣樹叢
- 樹冠較稀疏，能隱約看到林間幾束光柱（從左上 45 度照下）
- 主色：深藍綠 + 金色光柱

三張保持同色系、同筆觸，但密度與光感略不同。

每張 138×160 pointy-top hex，hex 邊外完全透明。
```

## 🔄 不滿意時的修圖指令

### hex 邊外有殘留色（不透明）
```
保持構圖跟畫風，但確認 hex 邊外的四個角是 100% 透明（不是白色、不是黑色）。重新產 PNG。
```

### 變體間色差太大（看起來像不同地形）
```
保持構圖，但讓 3 張的主色調再統一一點 — 用相同的色塊起手，只在細節（樹的位置、岩石分布）上差異化。
```

### 太陰沉 / 太亮
```
保持構圖跟筆觸，但把整體亮度調暗/提亮 15%，光源方向（左上 45 度）保持不變。
```

### 邊緣銳利不像 painterly
```
保持構圖跟色調，但讓邊緣更柔和、筆觸感更明顯 — 不要太接近 vector / clean line 的風格。
```

### 河流看起來太靜
```
保持構圖，但加強水流動感 — 波紋線條更明顯、白色高光分布更生動，要有「流動」而非「鏡面」感。
```
