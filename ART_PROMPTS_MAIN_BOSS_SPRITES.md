## 1️⃣ 伯朗將軍（Baron）— 王國終結者

```
BOSS 1：伯朗將軍。

維持 Q 版 JRPG 戰場 sprite 風格，但氣場更強、體格更壯。

設定:
- 性別年齡：男性，55 歲（Q 版下仍可看出老將的氣質）
- 髮型髮色：深灰色中分長髮夾雜白絲（Q 版下可簡化為白灰雙色塊），髮絲微微飄動
- 眼睛：冷酷的鋼灰色（Q 版下小眼睛、銳利眯起）
- 臉型：左眼斜向下到臉頰有明顯戰傷疤痕（Q 版下用一條斜紅線即可）
- 服裝：華麗銀色全板甲（肩甲特別誇張、有狼頭浮雕），胸甲有法因海姆狼頭紋章，黑紅雙色披風（從右肩披下，落地擺動感）
- 配件：右手扛著一把銀色巨型雙手劍（劍身扛在右肩上，劍長度約 1.5 倍身高，Q 版下可誇大）
- 姿勢：威懾站姿，左手插腰，重心後傾顯威壓感
- 表情：冷酷帶威脅微笑，露出尖齒
- 氣場：周身有淡灰色金屬寒氣（暗示「老將殺氣」）

256x256 透明背景，全身佔畫面 85% 高度，面向右側。
劍要明顯但不要超出 256x256 邊界（可斜放）。
```

## 🔄 不滿意時的修圖指令

### 氣場太弱（不像 BOSS）
```
保持構圖跟畫風，但讓周身的氣場光霧再明顯一點，讓人一眼看出是 BOSS 級別。
```

### 教主・黯太霸氣（應該平靜）
```
保持構圖，但把教主的姿勢改回完全放鬆，不要有戰鬥架式。重心居中、表情更平靜。要的是「智者反派」氣質。
```

### 影武者面罩看起來像普通頭盔
```
保持構圖，但把影武者的銀色金屬面罩畫得更精緻一點：要明顯是「下半臉的面罩」，能看到上方銀白頭髮和暗紅色左眼，下半臉只有面罩的金屬反光。
```

### 黑暗主教不夠詭異
```
保持構圖，但把眼下的三條黑色詛咒印記畫得更深一點，整體面色更蒼白，襯托邪教祭司的瘋狂感。
```

### 伯朗將軍劍太大壓住臉
```
保持構圖，但把巨劍稍微往右上斜放，讓劍身不要遮住臉。劍仍然要顯眼，但要看得到伯朗的表情。
```
