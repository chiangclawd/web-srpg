# Web Game — 西方奇幻 戰術 SRPG

> 個人開發的瀏覽器戰術角色扮演遊戲，靈感取自《天使帝國》、《火焰之紋章》、《三國志》系列。

## 📍 專案狀態

🟢 **10 章節完整故事 — 主線 8 章 + 番外 TRUE END 2 章**

主線（ch1-8）+ 番外（ch9-10 真·最終決戰）。**章節背景氛圍色**、**ESC 暫停選單**（動畫速度 / 靜音）、**localStorage 設定持久化**。

## 🚀 怎麼跑

### Web（主要）
```bash
npm install
npm run dev     # → http://localhost:5173
npm run build   # 型別檢查 + 打包到 dist/
```

### 桌面（Tauri 2，需先安裝 Rust + cargo）
```bash
# 一次性：安裝 Rust（rustup.rs）、安裝 tauri CLI
npm install
# 開發
npm run tauri:dev
# 打包安裝檔（macOS .dmg / Windows .msi / Linux .deb）
npm run tauri:build
```
設定檔位於 `src-tauri/`：`tauri.conf.json` / `Cargo.toml` / `src/main.rs`。

## 🎮 操作說明

- **標題**：「新遊戲」開新局；有存檔時顯示「繼續」與「清除存檔」
- **過場**：點擊或任意鍵推進對話
- **Hub（出陣前）**：點武將卡片看詳細資訊，按「出陣」進戰鬥
- **戰鬥**：
  - 點藍色我方單位 → 綠格移動 → 紅/橘/紫框攻擊（紫=被剋、橘=剋制）
  - 滑入敵人看**傷害預判 tooltip**
  - **攻擊有動畫**：近戰 lunge / 弓兵 projectile / 法師 magic burst
  - 受傷時跳出**傷害數字**
  - 「原地待命」結束行動，「結束我方回合」交給敵方
  - HP 0 → 撤退（不死亡）
  - 全滅一方 → 結算 → 自動存檔 → 下一章

## 📂 專案結構

```
src/
├── main.ts                 # Phaser 設定 + 場景註冊
├── BattleScene.ts          # 戰鬥主場景
├── Unit.ts                 # 武將單位類別（含攻擊動畫）
├── Grid.ts                 # 棋盤工具（Dijkstra/座標轉換）
├── types.ts                # 共用 TS 型別
├── data/
│   ├── commanders.ts       # 17 名武將
│   ├── unitTypes.ts        # 5 兵種
│   ├── equipment.ts        # 13 件裝備
│   ├── terrainTypes.ts     # 4 地形
│   ├── scenarios.ts        # 8 個戰場
│   ├── chapters.ts         # 8 章
│   ├── cutscenes.ts        # ~24 段對話腳本
│   ├── skillEffects.ts     # 14 個武將特技效果註冊
│   └── save.ts             # 存檔 load/save/clear
├── battle/
│   ├── CounterSystem.ts    # 兵種相剋（含飛兵）
│   ├── DamageCalculator.ts # 傷害計算（地形+特技+剋制）
│   └── Leveling.ts         # EXP/Level 常數
├── utils/
│   ├── audio.ts            # Web Audio 合成音效
│   ├── settings.ts         # 動畫速度/靜音/語言（localStorage）
│   ├── i18n.ts             # zh/en 雙語表 + t() 函式
│   └── hexDrawing.ts       # 六角形 path 繪製
└── scenes/
    ├── TitleScene.ts       # 標題（新遊戲/繼續/清除/語言切換）
    ├── CutsceneScene.ts    # 對話過場
    └── HubScene.ts         # 出陣前資訊（依章節篩武將）

src-tauri/                  # Tauri 2 桌面包設定
├── tauri.conf.json
├── Cargo.toml
├── build.rs
└── src/main.rs
```
