# 📜 剩餘章節 CG（Ch2 / 3 / 6 / 8 / 10）— Nano Banana Prompt

> 接續 `ART_PROMPTS_CHAPTER1_CG.md` 完成的 chapter1.png 之後，再做 5 張章節 CG。
> 程式已就緒：每張只要放到 `public/assets/cg/chapterN.png`，跟我說「chapterN 放好了」我就幫你開 manifest + push。

---

## 📐 共通規格（每張都要符合）

- **尺寸**：1920×1080 PNG（16:9 橫式）
- **檔名**：`chapterN.png`（N = 章節數）
- **路徑**：`public/assets/cg/chapterN.png`
- **構圖**：右下角留白給對話框（**畫面下方 1/3** 是對話框會壓住的區域，重要元素別放那）
- **使用方式**：cutscene 入場全螢幕背景 + 35% 黑罩 + 立繪坐在對話框上
- **無人物臉部、無文字、無浮水印**（角色用立繪）

---

## 🎬 共通風格錨定（新對話只貼一次，5 張共用）

```
我接下來要請你連續產出 5 張 JRPG 章節入場 CG，每張都要符合這個風格：

風格基準：
- 1920×1080 橫式構圖
- JRPG 章節入場大圖風格，類似 Tales of、Octopath Traveler 章節大圖
- 柔和賽璐珞著色 + 戲劇性光影
- 飽和度中等，色塊明確
- 電影感、手繪溫度，無 3D 渲染感
- 構圖原則：右下角留白給對話框（重要元素往中央和左半邊放）
- 無人物臉部、無文字、無浮水印

每張都要保持完全一致的線條粗細、上色技法、光影風格，這樣放在同一系列章節入場才不違和。

請確認你理解這個風格指引，接下來我會給你 5 個章節的詳細描述。

確認後請回我「了解，開始」，然後等我給第一張章節 CG。
```

等 Nano Banana 回「了解，開始」後，**才**開始貼下面的 prompt。

## 1️⃣ Chapter 2：黑松森（森林伏擊）

```
章節 2：黑松森伏擊。

主題：黑松森林——古樹遮天的伏擊地點 · 陰霾午後

構圖描述：
- 畫面前景（下方 1/3，左半邊）：石板小徑彎曲穿過森林中央，路面有腐葉與斷枝
- 畫面中景：兩側參天黑松巨樹（樹幹粗壯、樹皮深灰、樹冠濃密遮天），陽光從樹冠縫隙透下幾道斜射光柱
- 畫面遠景：森林深處的霧氣，模糊的樹幹剪影一直延伸到遠方
- 畫面上方：黑松樹冠交織覆蓋畫面頂部，只露出零碎的灰藍色天空
- 氛圍細節：地面瀰漫淡淡白霧，幾片落葉在光柱中飄落，路徑兩側看似有「視線」（但不畫具體人影，營造「有人在埋伏」的暗示感）

主色調：深綠、墨綠、灰白霧、零星金光（陽光斜射）
氛圍：壓迫、不祥、古老森林的沉重感，但又不是純黑暗——是「暴風雨前的寧靜」

避免：
- 不要畫角色或人臉
- 不要太亮，整體偏陰沉
- 不要直接畫敵人，只用「視線」氛圍暗示

輸出：1920×1080 PNG。
```

## 🔄 不滿意時的修圖指令

### 整體太亮 / 太暗
```
保持構圖跟畫風不變，把整體色調調暗 15%，但保持光源細節（陽光、月光、火焰）的對比度。
```

### 視覺重心不對
```
保持風格，但讓視覺重心往左偏移，右下角留更多空白（給對話框用）。
```

### Ch10 三契約者霧氣不夠暗示
```
保持構圖跟風格，把虛空裂縫旁邊的三縷紫黑霧氣畫得更明顯一點，要能讓人聯想到「三個人形輪廓」但不能太清楚。
```

### Ch6 哥德感不足
```
保持構圖，但讓哥德式石柱再雄偉一點、拱頂再陰森一點，紫黑契約之氣再多一點。
```

### Ch8 邪神之眼太搶眼
```
保持構圖，但讓邪神之眼更朦朧、更模糊，融入虛空背景，不要喧賓奪主。
```
